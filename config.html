Change Request App Configuration

This app uses the installation parameters configured during installation:

  • Freshservice Plan: Configures API rate limits based on your plan
  • API Safety Margin: Sets a safety margin to prevent hitting API limits
  • Inventory Software/Services Type ID: Determines which asset types to display
  • Search Cache Timeout: Controls how long search results are cached

To modify these settings, please reinstall the app or contact your administrator.
